📋 Visão Geral

O MotoRota é uma plataforma que conecta empresas que precisam de entregas com motoboys disponíveis, oferecendo rastreamento em tempo real, sistema de avaliações e gestão completa de pedidos.

🏢 Para Empresas

  • Cadastro com informações comerciais
  • Criação e gestão de pedidos
  • Acompanhamento em tempo real
  • Sistema de avaliação de motoboys
  • Relatórios e estatísticas
  • Configuração de raio de entrega

🏍️ Para Motoboys

  • Cadastro com documentação
  • Sistema de disponibilidade
  • Visualização de pedidos disponíveis
  • Gestão de entregas
  • Avaliação de empresas
  • Controle de ganhos e estatísticas

⚡ Sistema de Entregas

  • Rastreamento em tempo real
  • Múltiplos status de acompanhamento
  • Sistema de prioridades
  • Agendamento de entregas
  • Avaliações bidirecionais
  • Histórico completo

🔐 Autenticação

A API utiliza Auth0 para autenticação. Todas as rotas protegidas requerem um token JWT válido.

⚠️ Importante: Include o token no header Authorization: Bearer <seu-jwt-token>
GET /api/auth/config

Obter configurações do Auth0 para o frontend

GET /api/protected

Teste de autenticação - Rota protegida para validar tokens

🏢 Empresas

Gestão de Perfil

POST /api/companies/register

Registrar nova empresa no sistema

GET /api/companies/profile

Obter perfil da empresa autenticada

PUT /api/companies/profile

Atualizar dados do perfil da empresa

GET /api/companies/stats

Obter estatísticas e relatórios da empresa

Rotas Públicas

GET /api/companies

Listar empresas ativas (público)

GET /api/companies/:id

Obter dados públicos de uma empresa específica

🏍️ Motoboys

Gestão de Perfil

POST /api/motoboys/register

Registrar novo motoboy no sistema

GET /api/motoboys/profile

Obter perfil do motoboy autenticado

PATCH /api/motoboys/location

Atualizar localização atual do motoboy

PATCH /api/motoboys/availability

Alterar disponibilidade para receber pedidos

Para Empresas

GET /api/motoboys/available

Listar motoboys disponíveis por região

📦 Pedidos e Entregas

Gestão de Pedidos

POST /api/orders

Criar novo pedido (empresas)

GET /api/orders/company

Listar pedidos da empresa

GET /api/orders/motoboy

Listar pedidos do motoboy

GET /api/orders/available

Listar pedidos disponíveis (motoboys)

Ações de Entrega

POST /api/orders/:id/accept

Aceitar pedido (motoboys)

PATCH /api/orders/:id/status

Atualizar status da entrega

POST /api/orders/:id/rate-delivery

Avaliar entrega (empresas avaliam motoboys)

POST /api/orders/:id/rate-company

Avaliar empresa (motoboys avaliam empresas)

🔍 Rastreamento

GET /api/track/:trackingCode

Rastrear pedido por código (público - sem autenticação)

Status disponíveis:

  • pending - Pendente (aguardando motoboy)
  • accepted - Aceito por um motoboy
  • picked_up - Coletado no estabelecimento
  • in_transit - Em trânsito para entrega
  • delivered - Entregue com sucesso
  • cancelled - Cancelado

💻 Exemplos de Uso

1. Registrar Empresa

curl -X POST {{baseUrl}}/api/companies/register \ -H "Authorization: Bearer <jwt-token>" \ -H "Content-Type: application/json" \ -d '{ "name": "Pizzaria do João", "email": "contato@pizzaria.com", "phone": "(11) 99999-1111", "cnpj": "12.345.678/0001-90", "address": "Rua das Flores, 123, Centro, São Paulo, SP", "coordinates": { "lat": -23.5505, "lng": -46.6333 }, "businessType": "restaurante" }'

2. Criar Pedido

curl -X POST {{baseUrl}}/api/orders \ -H "Authorization: Bearer <jwt-token>" \ -H "Content-Type: application/json" \ -d '{ "customerName": "Ana Costa", "customerPhone": "(11) 99999-4444", "delivery": { "address": "Rua das Palmeiras, 321, Jardins, São Paulo, SP", "coordinates": { "lat": -23.5629, "lng": -46.6544 } }, "items": [{ "name": "Pizza Margherita Grande", "quantity": 1, "price": 32.00 }], "totalValue": 32.00, "paymentMethod": "card" }'

3. Rastrear Pedido

curl -X GET {{baseUrl}}/api/track/MR789456ABCD

4. Motoboy Aceitar Pedido

curl -X POST {{baseUrl}}/api/orders/123/accept \ -H "Authorization: Bearer <jwt-token-motoboy>"
